# On-Prem / Hybrid Cloud Architecture

## Overview
Typical service environments were operated across physical servers, virtualized infrastructure, and cloud resources when needed.

## Key Components
- Web layer: Nginx / Apache
- Load balancing: HAProxy / LVS
- Database: MySQL / MariaDB / PostgreSQL
- Virtualization: Proxmox / Xen / VMware
- Cloud: AWS / Azure

## Goals
- Stability
- Scalability
- Maintainability
- Operational efficiency
