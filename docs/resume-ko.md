# 백흥주 | DevOps / SRE / 플랫폼 엔지니어

## 프로필

20년 이상 인프라 운영 및 구축 경험을 보유한 엔지니어입니다.  
대규모 서비스 환경에서 안정적인 운영, 장애 대응, 자동화, 모니터링 체계 개선 업무를 수행해 왔습니다.  
온프레미스 및 클라우드 환경에서의 설계·운영 경험과 함께, Legacy 시스템의 가상화 및 현대화 경험을 보유하고 있습니다.

## 핵심 역량

- Linux / Windows 서버 운영 및 구축
- AWS / Azure / On-Prem 하이브리드 환경 운영
- Ansible / Shell 기반 운영 자동화
- Zabbix / Grafana 기반 모니터링 구축
- MySQL / MariaDB / PostgreSQL 운영
- HAProxy / Nginx 기반 서비스 운영
- 장애 대응 및 운영 안정화
- 가상화 및 Private Cloud 전환 경험
